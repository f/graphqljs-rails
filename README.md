# GraphQL.js for Rails

Adds [GraphQL.js](http://github.com/f/graphql.js) to your asset pipeline.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'graphqljs-rails'
```

And then execute:

    $ bundle install

And install it to your `application.js`:

    //= require graphql
